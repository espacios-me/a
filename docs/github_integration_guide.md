# GitHub Integration Guide\n\nThis guide explains how to set up and use the GitHub repository browser.\n\n## Setup\n1. Register a GitHub OAuth App.\n2. Set GITHUB_CLIENT_ID and GITHUB_CLIENT_SECRET in your worker environment.\n3. Deploy the worker to Cloudflare.\n\n## Usage\n1. Navigate to the Integrations tab.\n2. Click 'Connect GitHub'.\n3. After authorization, use the Repository Browser tab to search and filter your repos.
